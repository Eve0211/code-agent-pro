## Entry Points

### 1. SKILL.md — Main Entry

加载顺序：`skill.json` (触发匹配) → `SKILL.md` (完整指令) → `references/*.md` (按需)

### 2. SPEC Workflow (7 Phases)

```
Phase 1: Requirement Clarification     → 提问消除歧义
Phase 2: Generate SPEC Document        → 调用 write-a-prd skill
Phase 3: Architecture Design           → 3 paths (minimal/clean/pragmatic)
Phase 3.5: Task Decomposition          → 原子任务 + 验证方法
Phase 4: Implementation                → 用户确认后实施
Phase 5: Quality Gates                 → Lint → Type → Tests → Security
Phase 6: Self-Reflection               → "这是最优方案吗？"
```

### 3. `/learn` Command (6 Steps, ~30 min)

```
Step 1: Surface Scan (3 min)      → README + manifest
Step 2: Entry Point Trace (5 min) → main files + API routes
Step 3: Core Flow Deep Dive       → 1-2 main workflows
Step 4: Pattern Extraction        → code style + conventions
Step 5: Mental Model              → memory/project-map.md
Step 6: Gap Report                → targeted questions
```

### 4. Plan Mode (READ-ONLY)

规则：
- ❌ 无文件修改
- ❌ 无 git 操作

## Three-Layer Capability Model

| Layer | 内容 | 来源 |
|-------|------|------|
| Layer 1 | 通用模式（架构、错误处理、测试、安全） | 内嵌在 skill |
| Layer 2 | 框架专项（React 静默陷阱） | 外部 skill |
| Layer 3 | 项目规范（CLAUDE.md、ruff.toml） | 项目文件 |

**设计决策：React 有专项 skill，后端语言不需要**
- React 问题静默（重渲染通过所有检查）
- 后端问题透明（profiling 可发现）

## Version Compatibility Rules

**核心规则：永不编造 API 签名或注解。**

验证流程：
```
读取 manifest → 检测版本 → 不确定时搜索官方文档
```

高风险点（见 SKILL.md 完整表格）：
- Java/Spring: `javax.*` → `jakarta.*`, `WebSecurityConfigurerAdapter` removed
- Python: SQLAlchemy 1→2, Pydantic v1→v2, Django 4→5
- Node/TS: Next.js 13→15, React 18→19, Express 4→5
- Go: 泛型(1.18+), 循环变量作用域(1.22+)
- Rust: Edition 2021→2024, axum 0.6→0.7

npx code-agent-pro              # 安装到所有检测到的 Agent
npx code-agent-pro --claude     # 只装 Claude Code

## Key Design Decisions

1. **SPEC-first**：不经用户确认不得写代码
2. **Progressive disclosure**：SKILL.md < 500 行，详细内容在 references/
3. **Graceful degradation**：外部 skill 缺失不阻断工作流
4. **Three-layer model**：通用 → 框架专项 → 项目规范
5. **Plan Mode**：复杂任务先规划后实施
6. **Version compatibility**：永不编造 API，不确定时搜索文档
